# Student-Feedback-Management-System
# NAME :- TEJASHREE H
## DESCRIPTION :- This project is a Java-based desktop application designed to manage student feedback in a simple and organized way. The system allows users to add, update, delete, and view feedback records for different courses and teachers. It includes a clean interface where details like student name, course, teacher, rating, and comments can be entered and stored. A dynamic table displays all feedback entries for quick review.

 *Key Features*

 * User-friendly interface to manage student feedback details.

 * Add, update, and delete feedback through interactive buttons.

 * Displays all stored feedback in a structured data table.

 * Stores multiple fields such as student name, course, teacher name, rating, and comments.

 * Clean layout for easy input and data visibility.

 * Efficient for academic record-keeping and teacher performance review.
